Szyfrowanie tekstu

Wybierz Test lub Przesunięcie, wpisz tekst i liczbę całkowitą, a następnie kliknij przycisk Szyfruj.

Wybranie Test spowoduje użycie implementacji TestCoderImpl, która wyświetli wartości argumentów.

Wybranie Przesunięcie liter spowoduje użycie implementacji CoderImpl, która zwróci tekst przesuwając jego litery o zadaną wartość. Wartość liczbowa musi znajdować się w przedziale od 0 do 26.
